<div>Сервер сделан и проверен на Windows 10, на других ОС может работать некорректно.</div>
<div>Что необходимо для запуска:
<ul>
  <li>Microsoft Visual C++ Redistributable (требуется для работы многих программ, так что скорее всего уже установлен на компьютер)</li>
  <li>Node.js</li>
  <li>Python 3.12.4</li>
  <li>Python 3.11.6</li>
  <li>SPOTIFY CLIENT ID и SPOTIFY CLIENT SECRET (для их получения создайте своё приложение на сайте https://developer.spotify.com/dashboard)</li>
  <li>Ваш домен или внешний IP и открытый порт (по этому поводу обращайтесь к вашему WI-FI провайдеру)</li>
</ul>
</div>
<div>Чтобы запустить сервер достаточно запустить файл start.bat </div>
<div>При запуске:
<ol>
  <li>Произойдёт проверка на наличие Microsoft Visual C++ Redistributable</li>
  <li>Произойдёт проверка на наличие Python 3.12.4 и 3.11.6</li>
  <li>Создадутся виртуальные окружения для Python 3.12.4 и 3.11.6</li>
  <li>В виртуальные окружения установятся все необходимые библиотеки</li>
  <li>Вас потребуют ввести SPOTIFY CLIENT ID и SPOTIFY CLIENT SECRET</li>
  <li>Вас потребуют ввести ваш домен или внешний IP</li>
  <li>Вас потребуют ввести открытый порт на вашем роутере</li>
  <li>Произойдёт обновление всех парсеров (yt-dlp, spotdl)</li>
  <li>Сервер запущен!</li>
</ol>
</div>
<div>При повторных запусках шаги 3, 4, 5, 6, 7 будут пропущены </div>
